A simple blog built using djangae.

  • The blog component renders traditional django templates, making queries inside the appropriate view.
  • The feed component renders template snippets using Backbone, making queries to a basic API provided by the django rest framework.
  • The search component provides functionality to find articles which contain a keyword in their title/slug field.
  • The admin component allows authenticated users to create/edit/delete article objects using a custom interface.
  • The backend data is stored in the App Engine Datastore, a schemaless NoSQL database.
  • Authentication is possible using an approved Google account.
  • The project structure is taken from djangae-scaffold.
